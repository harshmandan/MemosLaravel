## Installation & Set-up

Requirements:
XAMPP or a similiar LEMP stack.
Clone this repository and put it in the ``htdocs`` folder.
Set up Apache ``vhost.conf`` and OS driver ``hosts`` file.
Create a database named "memo" using MySQL
Thats it!
